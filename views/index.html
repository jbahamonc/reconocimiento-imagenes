Reconocimiento de Imagenes con Visual Recoginition de IBM

SIRI es un sistema para la implementacion de reconocimiento de imagenes basado en la API que ofrece Visual Recognition de IBM.

1.Ir a Visual Recognition

Nos dirigimos a IBM Watson, una vez dentro nos dirigimos a Products&Service > Visual Recognition, donde nos redireccionara a la pagina de Recognition.
input

2. Crear cuenta en IBM Cloud

Una vez dentro de Visual Recognition, ingrese a la opcion Get Started Free y proceda a realizar la creación de la cuenta ingresando la informacion solicitada.
code

3. Crear proyecto de Visual Recognition

Una vez hallamos iniciado sesión, nos dirigimos a la barra superior izquierda para mostrar el panel y seleccionamos la opcion watson, luego seleccionamos la opcion crear recurso el cual nos dirigira a donde se encuentran todos los servicios, debe seleccionar Visual Recognition y luego Crear.
open_in_browser

4. Añadir el servicio de IBM Recognition

Una ves dentro del proyecto, procedemos a añadir el servicio de Recognition, para ello seleccionamos la opcion Getting Started Tutorial, luego seleccionamos la opcion Services que se encuentra en el paso 1 del tutorial. Finalmente dentro de los servicios buscamos Visual Recognition y seleccionamos Añadir servicio luego Crear proyecto.
code

5. Implementar Recognition en Node.js

Una vez dentro del servicio seleccionamos la opcion Referencia de api, allí nos dirigimos a la documentacion oficial de Visual Recognition. Para la implementacion solo basta con seguir las indicaciones de la documentación.
